# Lin

Save your terminal session's commands and output into a file with an ease.
What is saved:

- All commands and their outputs

## Installation

Please refer to:

- [linux](https://github.com/pacifiquem/lin.git#Linux) : For linux distros. ex: Ubuntu, Fedora ...
