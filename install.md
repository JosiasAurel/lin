# Lin

A tool that help you to save your terminal's session into a file with an ease.

## Linux

To install ``lin`` on any linux distro, do this:

1. Clone this repository: ```git clone https://github.com/pacifiquem/lin.git```.
2. ``make lin.out``
3. Run ``make install``.
